This application has been created for educational purposes and is available on GitHub.
It has explored using JavaScript, HTML and CSS to create a desktop application with Electron.

The application uses web scraping to retrieve the latest statistics from an existing website, the data is taken from Worldometers website.

The guidance on coronavirus are direct links to the NHS (National Health Service) webpages
